# Fontes de Dados — SELL INTELIGÊNCIA

Todas as fontes listadas são públicas ou abertas. Nenhum dado privado, de login ou sensível será coletado.

---

## Fontes Primárias (Contratos e Transparência)

| Fonte | Descrição | Acesso |
|-------|-----------|--------|
| **PNCP** | Portal Nacional de Contratações Públicas — contratos federais, estaduais e municipais | API pública |
| **Portais de Transparência Municipal** | Portais de cada prefeitura com execução orçamentária e contratos | Web scraping |
| **Diários Oficiais** | Publicações oficiais de contratos, licitações e atos municipais | Web scraping / PDF |
| **TCE-MS** | Tribunal de Contas do Estado — prestações de contas municipais | Consulta pública |

---

## Fontes Secundárias (Eventos e Contexto)

| Fonte | Descrição | Acesso |
|-------|-----------|--------|
| **Sites das Prefeituras** | Notícias, eventos e comunicados oficiais | Web scraping |
| **Redes Sociais Oficiais** | Facebook e Instagram das prefeituras (perfis públicos) | API pública / scraping |
| **Notícias Locais** | Portais regionais do MS (Campo Grande News, Correio do Estado, etc.) | Web scraping |
| **Google Trends** | Tendências de busca por artistas e eventos por região | API |
