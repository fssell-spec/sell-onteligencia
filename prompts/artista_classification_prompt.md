# Prompt: Classificação de Artistas

**Objetivo:** Classificar artistas mencionados em contratos e eventos quanto a gênero musical, faixa de cachê estimado, popularidade regional e compatibilidade com eventos municipais do interior do Mato Grosso do Sul.

**Campos a classificar:**
- Nome do artista
- Gênero musical principal: sertanejo / forró / pagode / funk / gospel / outro
- Subgênero (ex: sertanejo universitário, raiz, romântico)
- Faixa de cachê estimado: micro (<50k) / pequeno (50k-200k) / médio (200k-500k) / grande (>500k)
- Popularidade regional: local / estadual / nacional / nacional top
- Adequação a eventos municipais: alta / média / baixa
- Fontes consultadas: Spotify, YouTube, notícias de cachê

**Status:** Prompt a ser desenvolvido na etapa de base de artistas.
