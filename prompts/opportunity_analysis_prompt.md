# Prompt: Análise de Oportunidades Comerciais

**Objetivo:** Consolidar dados de prefeitura, histórico de eventos, contratos passados e tendências sazonais para gerar um parecer de oportunidade comercial e sugerir abordagem de vendas para a Sell Produtora.

**Entradas esperadas:**
- Perfil da prefeitura (população, orçamento estimado, histórico)
- Eventos realizados nos últimos 2 anos
- Contratos públicos relacionados
- Sazonalidade da região
- Artistas compatíveis disponíveis

**Saídas esperadas:**
- Score de oportunidade (0–100)
- Tipo de proposta recomendada: show avulso / rodeio completo / estrutura / pacote completo
- Faixa de orçamento provável
- Timing sugerido para abordagem
- Pontos de atenção e riscos

**Status:** Prompt a ser desenvolvido na etapa do motor de oportunidades.
