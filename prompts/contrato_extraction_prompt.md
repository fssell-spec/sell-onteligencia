# Prompt: Extração de Contratos Públicos

**Objetivo:** Extrair informações estruturadas de contratos públicos municipais relacionados a shows, rodeios, eventos e estruturas físicas a partir de textos de PDFs, diários oficiais ou portais de transparência.

**Campos a extrair:**
- Número do contrato
- Data de assinatura
- Vigência
- Valor total (R$)
- Nome do fornecedor / contratado
- CNPJ do fornecedor
- Objeto do contrato (descrição do serviço)
- Categoria inferida: show / rodeio / estrutura / produção / outros
- Artista mencionado (se houver)
- Município contratante
- Fonte do documento

**Status:** Prompt a ser desenvolvido na etapa de extração de PDFs.
