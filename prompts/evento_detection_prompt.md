# Prompt: Detecção de Eventos Públicos

**Objetivo:** Identificar e catalogar eventos públicos municipais (festas, rodeios, shows, aniversários de cidade) a partir de notícias locais, diários oficiais e redes sociais de prefeituras.

**Campos a extrair:**
- Nome do evento
- Município
- Data provável ou confirmada
- Tipo do evento: festa municipal / rodeio / show / aniversário / outro
- Artistas mencionados
- Estimativa de público
- Fonte da informação
- Grau de confiança: confirmado / provável / especulativo

**Status:** Prompt a ser desenvolvido na etapa de coleta de dados públicos.
